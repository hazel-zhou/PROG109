Classes I took at BC

Introduction to Web Development

Class Identification: PROG 109

Instructor: Adam Brehm

Course Description

Introduces fundamental programming techniques using current web development software. Students design procedures and write computer instructions to solve business problems, learn procedural programming, develop graphical user interfaces for the web and work with events and objects.

My Gain After This Class

I had zero knowledge about programming before taking this class. It's way more challenging than I imagine as I learn further. However, after taking this class, I have a basic understanding of programming, such as the basic elements of HTML, learning how to make a style for websites using CSS, and writing some basic JavaScripts.
